== Description ==

wp-shareto 插件方便访问者能分享or收藏你的文章！从而起到提高访客回头率、增加网站外链等。
相比用某些‘分享到’工具，它能更好的支持主页每篇文章的分享，不受外部网络的影响，只要你的博客可以访问。
现在提供的最新版也是唯一版本：1.0

相关链接:
* <a href="http://thobian.info/">晴天打雨伞</a>

<ol>
<li>Upload the folder `wp-shareto` to the `/wp-content/plugins/`directory</li>
<li>Activate the plugin through the 'Plugins' menu in WordPress</li>
<li>It worked</li>
